This is the project folder for the Gravitational N-Body problems project, by: Jeremy Bullock, Robert Burnet, and Cliff Vong.

./animations/ contains all the animations (mp4) files of the simulations.

./report/ contains the report itself as a pdf and its tex file, as well as the figures used in the report.

./combined_code.py is the entire code for our project, including the integration method solvers, test cases, and simulations.
